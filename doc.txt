--- REGISTER
POST /auth/register
BODY
  firstName
  lastName
  emailOrMobile
  password
  confirmPassword
RESPONSE
  200 message
  400 message
  500 message

--- LOGIN
POST /auth/login
BODY
  emailOrMobile
  password
RESPONSE
  200 accessToken
  400 message
  500 message

--- GET AUTHENTICATED USER
GET /auth/me
HEADER
  Authorization Bearer access_token
RESPONSE
  200 user
  400 message
  401 message
  500 message

--- UPLOAD PROFILE IMAGE OR COVER IMAGE
PATCH /users
HEADER
  Authorization Bearer access_token
BODY (multipart/form-data)
  profileImage
  coverImage
RESPONSE
  200 profileImage, coverImage
  400 message
  401 message
  500 message

--- GET USER PROFILE
GET /users/:profileId
HEADER
  Authorization Bearer access_token
RESPONSE
  200 profileUser
  400 message
  401 message
  500 message

--- REQUEST FRIEND
POST /relationships/users/:receiverId
HEADER
  Authorization Bearer access_token
RESPONSE
  200 message
  400 message
  401 message
  500 message
